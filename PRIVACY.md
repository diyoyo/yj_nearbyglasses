## 2. Principle

The protection of your data is important.
This app:

- **does not collect any personal data**
- does **not automatically transmit data to servers**
- does **not use an internet connection**
- does **not contain advertising**
- does **not use tracking or analysis services**
- does **not pass on data to third parties**

There is **no registration, no login, and no user tracking**.

## 3. How the app works

The app passively scans Bluetooth signals in the vicinity.

Only the following information is collected and displayed within the app:

- Time of the scan
- Signal strength (RSSI)
- Device name sent (if available)
- Manufacturer code sent
This is publicly available information from the BLE ADV frame and does not make you identifiable at all.

**Important:**

- **No active Bluetooth connection** is established with other devices. 
- **No communication with other devices** takes place. 
- Devices are detected passively only.

### 4.2 Personal data

**No personal data** is collected, stored or processed.

In particular, the following is **not recorded**:

- No name
- No email address
- No telephone number
- No location data
- No smartphone device identifiers
- No IP address
- No advertising ID
- No contacts
- No files or media content

### 4.2 Technical scan data
The app only processes technical signal information from Bluetooth devices within Bluetooth Low Energy (BLE) range.

This information:

- does not relate to the user's smartphone
- does not contain any information about the user
- is only processed locally on the user's device
 
## 5. Data transfer

Data is not automatically transferred to third parties.

The only way to transfer data is via the **Share function**, which is actively triggered by the user.
When this function is used, a text file is generated containing the following information:

- Time
- Signal strength
- Scanned device name (if available)
- Scanned manufacturer code

**No information about the user's own smartphone or the user as a person or entity** is transmitted.

The transfer takes place exclusively via the app selected by the user (e.g. email or messenger).
From this point on, the data protection regulations of the respective service apply and are not part of these regulations nor the App *Nearby Glasses*.

## 6. Internet usage

The app does **not have any function for communication via the Internet**.
**No data** is transferred to external servers.

## 7. Bluetooth permissions

The app requires Bluetooth permissions to scan devices in the vicinity.

These permissions are used exclusively for:

- passive detection of Bluetooth signals
- display of technical signal data within the app

There will be:

- **no active connection** established
- **no pairing** performed
- **no data sent** to other devices

## 8. Data security

As no data is transferred to external servers, there is no risk of server-side storage.
All processed information remains stored locally on the user's device and is subject to the security mechanisms of the Android operating system.

## 9. Data storage and deletion

All information stored within the app remains locally on the device.
When the app is uninstalled, all stored data is automatically deleted. Furthremore, the app contains a **delete function** which removes currently stored data imediately from the app.


## 10. User rights

As no personal data is processed outside the user's own device, there are no obligations to provide information, make corrections or delete data in the sense of personal data storage. 

If you have any questions about data protection, you can contact the person responsible at any time.

## 11. Changes to this privacy policy

This privacy policy may be amended in the event of changes to the app or legal requirements.
